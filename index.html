آموزش گیت و گیتهاب

گیت چیه؟

  • گیت ابزاریه که مدریت پروژه های ما رو به عهده میگیره و امانات زیر رو هم به ما میده:
  • - گیت امکان نگهداری و ذخیره کد های ما به صورت مرحله به مرحله رو داره
  • - گیت به افراد یه پروژه این قابلیت رو میده که بتونن با هم روی یک پروژه کار کنن
  • - گیت مدیریت کلی کد های ما رو انجام میده

چطوری بفهمیم گیت روی سیستم نصب هست یا نه؟

git --version
git version 2.34.1

نکته:

ما دستورات گیت رو توی محیط ترمینال سیستم عامل وارد میکنیم. اگه با ترمینال زیاد آشنا نیستید نگران نباشین، دستورات رو با هم بررسی میکنیم و دونه دونه میریم جلو

Terminal Inputes:

دستورات ترمینال به ورودی و خروجی های استاندارد یا SDI: Standard Input و SDU: Standard Output تقسیم میشن. ما دستورات ورودی رو از روی علامت شناسایی میکنیم. دستوراتی که اولشون این علامت هست رو میتونین کپی کنین و توی ترمینال سیستم تون وارد کنید و تست شون کنین. مثلا توی مثال بالا git --version یه دستور ورودیه و git version 2.34.1 یک خروجی استاندارد برای این دستور هست. اگه با هر پیامی غیر از این پیام رو به رو شدین احتمالا گیت روی سیتم تون نصب نیست و باید اون رو نصب کنین.
